# Express.js Assignment

## 1. Create a New Project

- Create an npm project.
- Install **Express.js**.
- *(Optional)* Install **Nodemon**.

---

## 2. Create an Express.js App

Create an Express.js application that:

- Funnels incoming requests through **two middleware functions**.
- Each middleware logs something to the console.
- Returns **one response** to the client.

---

## 3. Handle Routes

Handle requests to the following routes:

- `/`
- `/users`

Requirements:

- Each request should be handled by **only one** handler/middleware.
- Each handler should send a simple (dummy) response.
